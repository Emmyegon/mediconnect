# HealthLink - e-Clinic & Triage System

A comprehensive healthcare management system built with the MERN stack (MongoDB, Express.js, React, Node.js) that provides e-clinic and triage solutions for healthcare providers.

## Features

- **User Authentication** - Secure login and registration system
- **Patient Management** - Create and manage patient records
- **Appointment Scheduling** - Book and manage appointments
- **Triage System** - Prioritize patients based on medical urgency
- **Medical Records** - Digital record keeping for patients
- **Responsive Design** - Works on desktop and mobile devices

## Tech Stack

- **Frontend**: React, Redux Toolkit, TailwindCSS, HeadlessUI
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)
- **Rich Text Editor**: TipTap

## Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)
- MongoDB (local or Atlas)
